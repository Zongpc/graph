# Graph OP on AdjList

* compile quick start
    ```
    make clean
    make
    ```

* executable usage
    ```
    ./graph ../dataset/oregon1_010526.txt/oregon1_010526.txt 65400
    ```

* cross compile if you wanna run on arm 
    ```
    make clean
    make CC="aarch64-linux-gnu-gcc"
    ```
